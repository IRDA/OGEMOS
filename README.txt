# OGEMOS
Outil de gestion de la matière organique des sols basé sur le modèle ICBM (Introductory Carbon Balance Model)

Instructions pour télécharger l'exécutable le plus à jour:
 - Sélectionner et ouvrir le dossier Executables.
 - Télécharger le fichier zip contenant les exécutables et le guide d'utilisation. 
    Le bouton pour télécharger se trouve à droite (Download).
 - Décompresser le fichier sur votre ordi avant d'exécuter APP_OGEMOS.exe (avec interface GUI)
    ou API_OGEMOS.exe (sans interface graphique (GUI)) qui se trouve dans OGEMOS\APP_OGEMOS\
