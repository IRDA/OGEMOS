# OGEMOS
Outil de gestion de la matière organique des sols basé sur le modèle ICBM (Introductory Carbon Balance Model)

Instructions pour se procurer l'exécutable:
- En dessous de la navbar de github, il y a la navbar du projet. 
  En dessous de cette seconde navbar, il y a une combobox à gauche
  qui permet de changer la branche. Pour se procurer l'exécutable,
  il faut sélectionner la branche "Production". L'exécutable se
  trouve dans le dossier "Executables". Il suffit de télécharger
  le zip, le dézipper et de sélectionner le fichier APP_OGEMOS.exe.
